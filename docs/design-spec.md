# 背了么 App 设计说明文档

## 配色方案

### 浅色模式（Light Mode）
- **主色（Primary）**：#4F7CFF  
- **辅助色（Secondary）**：#6BCB77  
- **强调色（Accent）**：#FF9F43  

**说明：**
- 主色用于按钮、主要交互元素、进度条等核心视觉焦点。
- 辅助色用于学习成功、完成状态、正确反馈等。
- 强调色用于提醒、待复习、重要提示等。

**中性色：**
- 背景色：#F7F9FC  
- 卡片背景：#FFFFFF  
- 主文字：#1F2937  
- 次级文字：#6B7280  
- 分割线：#E5E7EB  

---

### 暗色模式（Dark Mode）
- **主色（Primary）**：#6D8CFF  
- **辅助色（Secondary）**：#7EDC8D  
- **强调色（Accent）**：#FFB86B  

**说明：**
暗色模式下主色适度提高亮度以保证对比度和可读性。

**中性色：**
- 背景色：#121212  
- 卡片背景：#1E1E1E  
- 主文字：#F3F4F6  
- 次级文字：#9CA3AF  
- 分割线：#2C2C2C  

---

## 字体

- **标题字体**：PingFang SC / 思源黑体 Bold  
- **正文字体**：PingFang SC / 思源黑体 Regular  

**字体层级：**
- 一级标题：22px / Bold  
- 二级标题：18px / Semibold  
- 正文：14px / Regular  
- 辅助信息：12px / Regular

# 核心页面

## 1. 首页

**页面定位：**  
用户每日学习入口，提供学习任务提醒和推荐内容。

**主要功能：**
- 今日学习任务概览
- 今日复习任务提醒
- 每日推荐单词
- 个人设置入口

**页面模块：**

1. 头像入口
2. 学习卡片
3. 复习卡片
4. 今日推荐单词

---

## 2. 闯关页面

**页面定位：**  
通过游戏化方式提升用户记忆效率与学习兴趣。

**主要功能：**

- 场次选择
- 单词对战
- 排行榜
- 闯关记录

**页面模块：**

1. 初级场
2. 中级场
3. 高级场
4. 排行榜
5. 闯关历史记录

---

## 3. 我的页面

**页面定位：**  
学习管理与数据展示中心。

**主要功能：**

- 当前学习词库管理
- 学习工具入口
- 学习数据统计

**页面模块：**

1. 当前词库
2. 学习工具
3. 学习数据仪表盘

---

# 交互说明

## 页面跳转

- 点击 **首页左上角头像** → 进入 **个人设置页面**
- 点击 **学习卡片** → 进入 **学习单词页面**
- 点击 **复习卡片** → 进入 **复习页面**
- 点击 **今日推荐单词** → 进入 **单词详情页**
- 点击 **加入生词本** → 添加至 **生词本**
- 点击 **闯关场次（初级/中级/高级）** → 进入 **闯关对战界面**
- 闯关结束 → 自动跳转 **闯关结果页**
- 点击 **排行榜** → 查看 **全服排行**
- 点击 **闯关记录** → 查看 **历史战绩与成就墙**
- 点击 **我的** → 进入**个人中心页面**
- 点击 **换本词书** → 进入 **词库选择页面**
- 点击 **生词本** → 查看 **收藏单词列表**
- 点击 **离线词库** → 查看 **本地词库下载与管理**

---

## 基础交互

- **下拉刷新**：  
  首页支持下拉刷新更新学习任务及推荐单词。

- **点击反馈**：  
  所有按钮提供点击态反馈（颜色加深或轻微缩放）。

- **进度条动画**：  
  学习进度条在进入页面时动态加载。

- **卡片点击效果**：  
  卡片点击产生轻微阴影变化与缩放效果。

---

## 反馈机制

- 正确回答：绿色提示 + 动画反馈  
- 错误回答：橙色提示 + 正确答案展示  
- 加入生词本：Toast提示“已加入生词本”  

---

## 适配说明

- 支持 **浅色模式 / 深色模式自动切换**
- 支持 **不同屏幕尺寸自适应布局**
- 支持 **字体大小调节**
