## 心得体会

这次 AI 集成让我体会到，工程落地的重点不只是"模型能生成内容"，更是"输出可稳定解析、服务可观测、异常可定位"。在实际开发中遇到了 AI 返回非标准 JSON 格式的问题（如被代码块包裹、包含尾逗号等），通过增加容错解析逻辑解决了这些问题。

此外，通过补充日志和容错解析，我对后端 AI 接口的稳定性设计有了更深理解，也认识到环境变量管理 API Key 的重要性。这些经验对我今后的后端开发很有帮助。
