## 遇到的问题和解决

1. **问题：学习记录未正确写入**

   * 解决：在 `submitLearnResult` 方法中增加每日学习记录更新逻辑

2. **问题：JWT 用户信息获取不统一**

   * 解决：封装工具方法统一从请求中解析用户 ID

3. **问题：MyBatis 无法映射 JSON 字段**

   * 解决：使用自定义 TypeHandler 处理 JSON 与对象转换

4. **问题：测试用例与接口文档不一致**

   * 解决：逐一修复测试用例，确保与接口文档保持一致
