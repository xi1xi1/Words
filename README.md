## 项目简介

**“背了么”** 是一款面向大学生的轻量级背单词 App，旨在以更低门槛、更有趣的方式，解决“背单词难以坚持”的问题。

---

## 核心功能

| 模式 | 功能描述 |
|------|------|
| **标准背词** | 卡片式学习，支持释义、发音、拼写测试，满足日常积累 |
| **分级闯关** | 分设**初级场/中级场/高级场**，通关即可获得积分，**场次越高，单局积分越多** |
| **全服排行榜** | 所有用户的积分汇总排名，**看得见大家，但没有实时对战压力** |
| **AI词本** | 用户添加生词后，AI自动生成原创例句和场景对话，帮助理解用法 |

---

## 目标用户

- **在校大学生**（四六级、考研备考）
- **背单词总放弃的“反复开始者”**
